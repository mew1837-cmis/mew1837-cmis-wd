• Steps to creating a website in sftp
  • Open terminal and clone directory
  • Cd into userid-cmis-wd using the command "cd userid-cmis-wd"
  • Ensure you have the completed html and css files in your repository (userid-cmis-wd)
  • Validate filename.html using https://validator.w3.org/
  • Validate filename.css using https://jigsaw.w3.org/css-validator/
  • Log into sftp with userid@cmis-hsteacher from your repository folder
  • Cd into public_html using the command "cd public_html"
  • Put html and css files using the command "put filename.html" and "put filename.css"
