Steps to creating a website on the production server

  1. Open terminal and clone directory
  2. Cd into userid-cmis-wd using the command "cd userid-cmis-wd"
  3. Save a copy of your template.html as filename.html using gedit. Add content as required.
  4. Ensure you have the completed html and css files in your repository (userid-cmis-wd)
  5. Validate filename.html using HTML Validator
  6. Validate filename.css using CSS Validator
  7. Log into sftp with userid@cmis-hsteacher from your repository folder
  8. Cd into public_html using the command "cd public_html"
  9. Put html and css files using the command "put filename.html" and "put filename.css"
  10. Ensure website is valid by navigating to http://cmis-hsteacher/~userid/filename.html
